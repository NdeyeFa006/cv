# Portfolio Website

Site statique créé pour Ndeye Fatou Ndiaye.

## Fichiers
- `index.html` : page principale
- `styles.css` : styles et animations
- `script.js` : menu mobile et animations de scroll

## Déploiement
Vous pouvez mettre ce site en ligne en publiant ces fichiers sur un hébergeur statique tel que GitHub Pages, Netlify ou Vercel.

### GitHub Pages
1. Créez un dépôt GitHub.
2. Poussez ces fichiers dans le dépôt.
3. Activez GitHub Pages dans les paramètres du dépôt en choisissant la branche `main` ou `master`.
4. Le site sera accessible en ligne via l'URL fournie par GitHub Pages.
